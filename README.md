# 💼 EXPENSE MANAGEMENT SYSTEM

A FULL-STACK APP TO AUTOMATE EXPENSE REIMBURSEMENTS WITH STRUCTURED APPROVAL WORKFLOWS AND OCR-BASED RECEIPT SCANNING.

---

## 🚀 TECH STACK

* **FRONTEND:** REACT (VITE), TAILWIND CSS
* **BACKEND:** FASTAPI (PYTHON)
* **DATABASE:** MYSQL
* **OCR:** TESSERACT
* **APIs:**

  * COUNTRIES / CURRENCY
  * CURRENCY CONVERSION

---

## 📌 FEATURES

### 🔐 AUTHENTICATION & USERS

* SIGNUP CREATES COMPANY + ADMIN (WITH DEFAULT CURRENCY)
* ADMIN CAN:

  * ADD EMPLOYEES / MANAGERS
  * ASSIGN ROLES
  * SET REPORTING HIERARCHY

---

### 💸 EXPENSE SUBMISSION

* EMPLOYEES CAN:

  * SUBMIT EXPENSES (AMOUNT, CATEGORY, DESCRIPTION, DATE)
  * USE DIFFERENT CURRENCIES
  * TRACK STATUS (PENDING / APPROVED / REJECTED)

---

### ✅ APPROVAL WORKFLOW

* MULTI-LEVEL APPROVALS (MANAGER → ADMIN)
* SEQUENTIAL FLOW (NEXT STEP TRIGGERS AFTER APPROVAL)
* APPROVERS CAN APPROVE / REJECT WITH COMMENTS

---

### ⚙️ CONDITIONAL APPROVALS

* PERCENTAGE RULE (E.G., 60% APPROVALS)
* SPECIFIC APPROVER RULE (E.G., ADMIN APPROVAL)
* HYBRID COMBINATIONS SUPPORTED

---

### 👥 ROLES

| ROLE     | ACCESS                                                |
| -------- | ----------------------------------------------------- |
| ADMIN    | MANAGE USERS, RULES, ALL EXPENSES, OVERRIDE APPROVALS |
| MANAGER  | APPROVE/REJECT, VIEW TEAM EXPENSES                    |
| EMPLOYEE | SUBMIT & TRACK EXPENSES                               |

---

### 📸 OCR

* UPLOAD RECEIPT → AUTO-EXTRACT:

  * AMOUNT, DATE, DESCRIPTION, VENDOR
* AUTOFILLS EXPENSE FORM

---

## 🗄️ DATABASE SETUP

1. OPEN MYSQL WORKBENCH
2. GO TO **SERVER → DATA IMPORT**
3. SELECT `app/db.sql`
4. IMPORT INTO ANY SCHEMA
5. RUN IMPORT
